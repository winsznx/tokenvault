# TokenVault

Multi-asset custody system with flexible timelocks on Base and Stacks blockchains.

## Features

- Deposit any token with custom lockup period
- Time-based unlock mechanism
- Multiple deposits per user
- Secure withdrawal after timelock expiry
- Track all deposits with unique IDs

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Base**: Solidity 0.8.20, Foundry, Reown wallet
- **Stacks**: Clarity v4, Clarinet, @stacks/connect

## Getting Started

```bash
pnpm install
pnpm dev
```
